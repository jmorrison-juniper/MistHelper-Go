## Reporting a Vulnerability

If you discover a security issue, please report it privately:

**Email**: [jmorrison@juniper.net](mailto:jmorrison@juniper.net)

Include:
- Description of the vulnerability
- Steps to reproduce
- Affected version (commit hash or release tag)

**Please do not open a public GitHub issue for security vulnerabilities.**

## Response

This is a single-maintainer project. There is no SLA or guaranteed
response time. I will do my best to acknowledge reports promptly and
address confirmed vulnerabilities as time permits.

## Scope

This policy covers the MistHelper-Go application and its container
image (`ghcr.io/jmorrison-juniper/misthelper-go`). It does not cover the
Juniper Mist Cloud platform or the `mistapi-go` SDK.
